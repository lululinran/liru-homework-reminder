先安装 Python 3.8+ 和 Playwright 浏览器引擎（只需运行一次）。

git clone https://github.com/lululinran/liru-homework-reminder.git cd liru-homework-reminder pip install playwright playwright install chromium

配置微信推送（Server酱）

零成本把提醒推到微信，只需扫码两步。

# 1. 打开 https://sct.ftqq.com/ 微信扫码登录 # 2. 复制 SendKey，然后： cp config.json.example config.json # 编辑 config.json，填入 sendkey
💡 微信关注「方糖」公众号，推送消息会发到这里。

设置每日自动提醒

一键安装定时任务，每天早上 8:00 自动运行，无需手动操作。

# 先手动登录一次（保存 Cookie） python3 fetch.py python3 report.py # 测试推送是否正常 # 安装定时任务 chmod +x setup_launchd.sh ./setup_launchd.sh install # 管理命令 ./setup_launchd.sh status # 查看状态 ./setup_launchd.sh run # 立即运行一次 ./setup_launchd.sh uninstall # 取消定时
⚠️ Cookie 有效期通常为几周，过期后重新运行 python3 fetch.py 即可。

设置每日自动提醒

以管理员身份打开 PowerShell，运行安装脚本。

# 先手动登录一次（保存 Cookie） python fetch.py python report.py # 以管理员身份打开 PowerShell，进入目录后： .\setup_task.ps1 install # 安装定时任务 .\setup_task.ps1 status # 查看状态 .\setup_task.ps1 run # 立即运行一次 .\setup_task.ps1 uninstall # 取消定时
⚠️ 如希望锁屏时也能运行，打开「任务计划程序」→ 找到 LiruHomeworkReminder → 属性 → 安全选项 → 选择"无论用户是否登录都运行"。

🍪 关于 Cookie 自动登录

首次运行 python fetch.py 会弹出浏览器让你手动登录，登录成功后 Cookie 自动保存到本地。
之后定时任务使用 --headless 无头模式自动运行，不需要弹出浏览器。
Cookie 过期（通常几周后）只需再手动运行一次 python fetch.py 重新登录即可。
